# To-Do App

A simple To-Do app built with Flutter.

## Features
- Add tasks
- Mark tasks as complete
- Delete tasks

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/todo-app.git
   ```

2. Navigate into the project directory:
   ```bash
   cd todo-app
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
